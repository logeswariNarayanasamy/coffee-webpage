css display properties

explain blockdisplay: inline; display: block; block level element does not continuously display, does accept height, width,padding and margin and take a full with and height

explain inline display: inline; inline level element do continuously display , does not accept height, width, does accept padding and margin and this is not take a full with and height base on value of content,Element takes only required width and does not start on new line. Element takes full width and starts on a new line

display: inline-block; Element stays inline but allows width and height, margin, padding and does not start on new line. Link

display: none; Element is hidden and not displayed. remove the element from the document flow, and it will not take up any space on the page. This is different from visibility: hidden, which hides the element but still takes up space in the layout.
